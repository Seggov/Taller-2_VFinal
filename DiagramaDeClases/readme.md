# Diagramas del Proyecto 📊

Este directorio contiene el diseño de la arquitectura del sistema y el modelo de clases para el simulador de Pokémon.

---

## 📂 Archivos Disponibles

* **`Pokemon.drawio.jpg`**: Una exportación en imagen de alta resolución para visualizar el diagrama de clases UML de forma rápida y directa.
* **`Pokemon.drawio`**: El archivo fuente del diagrama. Puedes importarlo directamente en [draw.io (diagrams.net)](https://app.diagrams.net/) para editarlo, modificar las relaciones o añadir nuevas clases al diseño.
